# Expense-tracker-AI
AI-powered Expense Tracker web application built with HTML, CSS, JavaScript, and Chart.js. Features expense management, OCR receipt scanning, AI-based categorization, spending analytics, budget tracking, savings goals, multi-currency support, and interactive dashboards with a responsive modern UI.
# 💰 Expense Tracker AI

An AI-powered Expense Tracker web application designed to help users manage their finances efficiently. The platform provides expense tracking, spending analytics, budget monitoring, savings goals, and AI-driven financial insights through an intuitive and responsive interface.

## 🚀 Features

* 📊 Interactive Financial Dashboard
* 💵 Expense & Income Management
* 🧾 OCR Receipt Scanner Interface
* 🤖 AI-Based Transaction Categorization
* 📈 Spending Analytics & Forecasting
* 🎯 Budget Planning & Alerts
* 💰 Savings Goal Tracking
* 🔄 Recurring Expense Detection
* 🌎 Multi-Currency Support
* 📄 PDF & CSV Report Export
* 🌙 Dark/Light Mode
* 📱 Fully Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* Local Storage
